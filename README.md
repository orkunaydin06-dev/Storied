## Critical Rules

- **Privacy first.** Audio is sacred. Never used for AI training, never for marketing case studies.
- **Stoic, not cheerleader.** No exclamation marks in UI. No emojis. No "you got this!"
- **One practice per calendar day.** Strict. The spacing effect makes the learning real.
- **Day 30 must feel earned.** Quiet, weighty, no celebration emojis.
